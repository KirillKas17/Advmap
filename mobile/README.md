# Travel Game Mobile App

Мобильное приложение для платформы геймификации путешествий.

## Установка

```bash
cd mobile
npm install
```

## Запуск

### iOS
```bash
npm run ios
```

### Android
```bash
npm run android
```

## Структура

- `src/` - Исходный код приложения
  - `screens/` - Экраны приложения
  - `navigation/` - Навигация
  - `services/` - Сервисы (API, геолокация)
  - `contexts/` - React контексты
  - `config/` - Конфигурация
